# practice-express-mongo-blog
My practice of a blog system with express+mongodb

How to use:
1. Install dependencies:
```
cd <project-name>
  
npm install
```
2. Start:
```
npm start
```

If not working properly, try deleting:
package-lock.json file
node_modules folder,
and npm install again.
